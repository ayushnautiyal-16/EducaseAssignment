## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation & Development

**1. Install dependencies:**

```bash
npm install
```

**2. Start the development server:**

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

**3. Build for production:**

```bash
npm run build
```

**4. Preview the production build:**

```bash
npm run preview
```
